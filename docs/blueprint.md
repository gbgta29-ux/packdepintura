# **App Name**: WallTransform

## Core Features:

- VSL Player: Plays a video sales letter to engage visitors.
- Downloadable Guides: Allows users to download guides on wall decoration techniques.
- Bonus Unlocking: Dynamically unlocks bonus content based on user interaction and timers.
- Testimonial Display: Shows user testimonials as WhatsApp message simulations to build trust.

## Style Guidelines:

- Primary color: Gold (#D4AF37) to convey luxury and transformation.
- Background color: Pearl white (#F8F8FF) to give a sense of marbled clarity.
- Accent color: Lilac (#C8A2C8) to complement the gold, adding elegance.
- Headline font: 'Playfair', a modern sans-serif similar to Didot, for a high-end feel; body font: 'PT Sans', a humanist sans-serif, for longer text.
- Use decorative icons related to interior design and art.
- Use a clean layout with clear section breaks, mimicking a sales page structure.
- Subtle animations to highlight call-to-action buttons and create a sense of urgency.
